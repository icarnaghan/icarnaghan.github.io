How to increase the font size for Windows XP

Open the Display Options by right-clicking on a open spot on your Desktop and then select Properties. Click on the Appearance Tab. In the Font Size dropdown box select the appropriate font size. Once satisfied click on OK to save the changes and close the Display Options.
